# UI截图到配置文件映射系统

## 目录说明

这个目录用于存储UI元素到配置文件的映射数据库，以及相关的查找流程文档。

## 文件结构

- `mapping_db.json` - 映射数据库，记录UI元素特征与配置文件的对应关系
- `lookup_guide.md` - 查找流程指南
- `examples/` - 示例截图和对应的映射示例

## 使用方式

当你发送截图（特别是带有红框标注的截图）时，AI会自动：
1. 分析截图中的UI元素（文本、位置、布局）
2. 在 `mapping_db.json` 中查找匹配项
3. 返回对应的配置文件路径

## 添加新映射

当遇到新的UI元素时，可以在 `mapping_db.json` 中添加新的映射项。
